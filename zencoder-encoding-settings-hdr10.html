Zencoder Encoding Settings: HDR10

This topic contains details for encoding settings related to HDR10 video outputs.

Zencoder documentation has moved to provide you a more focused site. The new location is https://zencoder.support.brightcove.com. Please use the new site and update your bookmarks! These old docs will be removed soon.

Introduction

Zencoder supports high-dynamic-range (HDR) video, initially adding options to produce HDR10 outputs from HDR color-graded masters.

In addition to this topic, you should also look at HDR10 Video Output for additional information and specific video encoding settings that are required for HDR10 outputs.

preserve_hdr_metadata

master_display:Boolean

API Versions: V2

Parent: outputs

Default: false

Compatible Job Types: vod

Description:

If all the requirerd HDR10 metadata is in the source video, set preserve_hdr_metadata to true. The HDR10 metadata will be passed through to the encoder, and you will not need to include master_display and light_level settings listed below.

{
  "input": "s3://mybucket/hdr_master.mov",
  "outputs": [
    {
      "video_codec": "hevc",
      "video_bit_depth": 10,
      "preserve_hdr_metadata": true
    }
  ]
}

master_display

master_display:Object

API Versions: V2

Parent: outputs

Default: none

Compatible Job Types: vod

Description:

Hash containing color volume and luminance settings for HDR outputs.

{
  "input": "s3://mybucket/hdr_master.mov",
  "outputs": [
    {
      "video_codec": "hevc",
      "video_bit_depth": 10,
      "master_display": {
        "red": {
          "x": 0.708,
          "y": 0.292
        },
        "green": {
          "x": 0.17,
          "y": 0.797
        },
        "blue": {
          "x": 0.131,
          "y": 0.046
        },
        "white_point": {
          "x": 0.31273,
          "y": 0.329
        },
        "luminance": {
          "max": 1000,
          "min": 0.0001
        }
      },
      "max_content_light_level": 1000,
      "max_frame_average_light_level": 180
    }
  ]
}

max_frame_average_light_level

max_frame_average_light_level:Number

API Versions: V2

Parent: outputs

Default: none

Valid Values: 5.0-10000.0

Compatible Job Types: vod

Example: 180

Description:

Maximum average light level for content frames nits or candelas per square meter (cd/m2) - integer values. Although values up to 10000 are allowed, values in the 100-200 range will generally produce the best results.
